Brief explanation of Flexbox

Flexbox (Flexible Box Layout) is a CSS layout model designed to make it easier to align, distribute, and space items within a containerâ€”even when their sizes are unknown or dynamic. It allows elements to automatically adjust and organize themselves in a row or column, making web layouts more responsive, flexible, and easier to manage than older methods like floats or inline-block.
