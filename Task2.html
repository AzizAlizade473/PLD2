Aligning Elements

Flexbox Axes:

  • Main Axis → Defined by flex-direction (row by default = horizontal)
  • Cross Axis → Perpendicular to the main axis (vertical if flex-direction: row)

Horizontal Alignment (Main Axis)

Use justify-content:

  • flex-start – items align to the start (default)
  • center – items align to the center
  • flex-end – items align to the end
  • space-between – space between items
  • space-around / space-evenly – equal spacing

Vertical Alignment (Cross Axis)

Use align-items:

  • flex-start – align items to the top
  • center – align items vertically centered
  • flex-end – align items to the bottom
  • stretch – stretch items to fill container height (default)
