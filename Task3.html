Flex Axes and Properties

Difference Between Main Axis and Cross Axis in Flexbox:

In Flexbox, layout behavior is based on two axes:

Main Axis

  • The primary direction in which flex items are laid out.
  • Controlled by the flex-direction property.
  • Default: row â†’ items go left to right (horizontal axis).
  • Items are aligned along the main axis using justify-content.

Cross Axis

  • The perpendicular direction to the main axis.
  • Automatically set opposite to flex-direction.
  • Items are aligned along the cross axis using align-items or align-self.
