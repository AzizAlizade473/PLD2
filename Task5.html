flex Shorthand Property in CSS

The Flex shorthand sets three properties on a flex item:

  • flex: flex-grow flex-shrink flex-basis ;

Example:

flex: 1 0 100px;

  • 1 → Item can grow to fill space
  • Item will not shrink
  • Item starts at 100px width

Common Shortcuts:

  • flex: 1 → Grow equally, shrink if needed, basis is 0
  • flex: auto → Grow/shrink based on content size
  • flex: none → No grow, no shrink, size stays fixed
